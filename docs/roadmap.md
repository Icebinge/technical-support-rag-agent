# Learning Roadmap

## Week 1: Dataset And Retrieval Basics

- Download datasets.
- Verify fields and corpus coverage.
- Implement BM25 retrieval.
- Evaluate retrieval hit rate.

## Week 2: Dense Retrieval

- Use a local sentence embedding model.
- Build a vector index.
- Compare BM25 and dense retrieval.
- Write retrieval error analysis.

## Week 3: RAG Answering

- Generate answers from retrieved contexts.
- Add source citations.
- Detect unanswerable questions.
- Record hallucination and citation failures.

## Week 4: Agent Workflow

- Move the pipeline into LangGraph.
- Add query rewrite and retrieval grading.
- Add trace logging.
- Compare workflow variants.

## Week 5: API And Evaluation Report

- Build a FastAPI service.
- Add evaluation command.
- Generate a final report.
- Prepare resume bullets and interview explanation.
